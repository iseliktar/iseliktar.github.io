# SpoDari Web

This project runs on [Jekyll](https://jekyllrb.com/) and is hosted with [Github Pages](https://pages.github.com/).

## Dependencies:

* Jekyll:

`$ gem install bundler jekyll`


* Serve the content:

`$  bundle exec jekyll serve`
